# @himenon/datadog-typescript-openapi

```bash
# yarn
yarn add -D @himenon/datadog-typescript-openapi

# pnpm
pnpm add -D @himenon/datadog-typescript-openapi
```

## Usage

```ts
import { Client } from "@himenon/datadog-typescript-openapi/v1.22.3";
```

## Build

```ts
pnpm build
```
